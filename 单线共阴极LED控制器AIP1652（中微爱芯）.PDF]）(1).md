# AiP1652 单线串口LED驱动芯片手册（中微爱芯，版本2024-08-A1）

## 一、产品基础信息

### 1.1 芯片概述

AiP1652 为**共阴极LED专用驱动**，内置单线UART串口、上电复位、显示RAM；

- 输出：7路固定SEG段+5路固定GRID位 + 1路SEG8/GRID6复用端口
- 两种驱动组态：**8段×5位数码 / 7段×6位数码**（由GS配置位选择）
- 供电：3.0V\~5.0V；典型通信波特率19.2Kbps(17.5\~21.2Kbps区间自适应)
- 封装：SOP16

### 1.2 料号订购参数

| AiP1652SA16.TR | SOP16 | AiP1652 | 4000PCS/盘，8000PCS/盒         |

> 封装本体尺寸：10.0mm×3.9mm，引脚间距1.27mm

## 四、单线UART通信协议

### 4.1 单字节帧格式（固定11bit：1起始+8数据+1校验+1停止，**低位先发D0\~D7**）

1. **空闲态**：SDA持续高电平≥11bit时长
2. **起始位**：空闲后拉低SDA，保持1bit
3. **数据位**：D0\~D7共8bit，低位在先
4. **奇偶校验位**：D0\~D7累加和为奇数→校验0；偶数→校验1
5. **停止位**：拉高SDA保持1bit，随后回到空闲

### 4.2 两条核心指令

#### 指令1：写显示数据

> 首字节：`AD2 AD1 AD0 0 1 0 0 0`；AD\[2:0]=目标GRID位地址(0\~5)
> 后续N字节：S8\~S1，1字节对应1个数码段数据；**bit=1对应LED点亮，bit=0熄灭**
> 规则：单条指令全部数据发送完成后，SDA空闲>3ms自动锁存数据至DDRAM刷新显示；字节间间隔＜0.5ms

#### 指令2：芯片配置指令

| 字节     | bit7\~bit0           | 说明                          |
| ------ | -------------------- | --------------------------- |
| 配置首字节  | `00011000(0x18)`     | 进入配置模式固定指令码                 |
| 配置第二字节 | P3 P2 P1 P0 1 1 1 GS | GS：复用脚配置；P\[3:0]：全局PWM调光占空比 |

- GS位：`GS=0 → SEG8/GRID6作为SEG8（8段5位模式）`；`GS=1 → SEG8/GRID6作为GRID6（7段6位模式）`
- PWM调光(P3\~P0)：0000=全灭(0/16)、1000=1/16、1010=4/16、0110=4/16、1110=10/16；非0值开启显示

### 4.3 两种初始化流程（文字替代原图流程图）

#### 流程A：8段5位配置(GS=0)

1. 上电等待电源稳定
2. 发送写指令首字节`0x08(AD=0，GRID1)`，分5次依次写入GRID1\~GRID5显示数据，**单字节间隔<0.5ms**
3. 全部数据发完，保持总线空闲>3ms，数据存入显存
4. 发送配置首字节0x18 + 配置字节（GS=0+目标PWM值），完成亮度+模式配置

#### 流程B：7段6位配置(GS=1)

1. 上电等待电源稳定
2. 依次发送`0x08(GRID1)、0x88(GRID2)、0xA8(GRID3)…`，逐个点位写入显示数据，字节间隔<0.5ms
3. 全部写完空闲>3ms锁存
4. 配置指令0x18+配置字节(GS=1+PWM)，开启6位7段显示

## 五、典型应用说明

1. **应用1（8段×5位共阴数码）**
   SEG1\~SEG7+SEG8作为8段，GRID1\~GRID5作为5个位；SEG8/GRID6配置为SEG(GS=0)，数码公共阴极接GRID，段阳极接SEG串限流电阻。
2. **应用2（7段×6位共阴数码）**
   SEG1\~SEG7作为7段，GRID1\~GRID5+GRID6(复用脚)组成6个位；GS=1，SEG8/GRID6切换为位输出。

> MCU仅需**1根通用IO接SDA**即可完成全部驱动控制，VDD侧常规搭配104滤波电容。

## 七、使用备注

1. 芯片无硬件复位，上电显存数据随机，**建议先全写初始化显存再配置亮度**，避免上电闪乱码；
2. 指令内部字节间隔≤0.5ms，指令之间必须≥3ms空闲；
