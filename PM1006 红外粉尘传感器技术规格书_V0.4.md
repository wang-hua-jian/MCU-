# PM1006 红外粉尘传感器规格说明书
## 一、产品概述
### 1.1 工作原理
采用光学散射原理，红外LED发射光线遇空气中粉尘产生散射光，光敏器件接收散射光转为电信号；模块内置MCU、自带温度补偿运算，最终通过UART/PWM输出颗粒物质量浓度，检测粒径范围0.3μm~10μm。
### 1.2 产品优势
1. 内置MCU，直接输出颗粒物浓度数值；
2. 自带板载温度传感器，全温域温度补偿；
3. 电磁屏蔽结构，抗电磁干扰；体积小巧、低功耗、响应速度快、测量量程宽。
### 1.3 适用场景
空气净化器、车载净化器、家用/商用空调、新风系统、桌面空气质量检测仪、环境监测设备、IoT智能硬件。

## 四、通讯协议
### 4.1 UART串口通讯
- 串口配置：9600bps，8数据位，1停止位，无校验，4.5V电平
- 主机读取指令：`11 02 0B 01 E1`
- 设备应答帧：`16 11 0B DF1~DF16 + 校验CS`
- PM2.5浓度计算公式：`PM2.5 = DF3 × 256 + DF4`

### 4.2 PWM输出协议
1. 周期固定1000ms，低电平有效；
2. 浓度计算公式：`粉尘浓度(μg/m³)=低电平持续时间(ms)-4`
3. 量程约束：
   - 低电平4ms → 0μg/m³
   - 低电平996ms及以上 → ≥992μg/m³

## 五、使用规范与注意事项
1. 安装位置预留通风进气通道，保证气流正常流入检测腔；
2. 禁止在有机挥发气体、可燃气体环境中长期使用；腔体积灰可使用吸尘器清理；
3. 远离高频、高压元器件，避免电磁干扰；
4. 结构封装需避光，外界直射光会加大测量误差；
5. 避免整机剧烈震动，震动会造成测量精度变差。
