# ZS03 单总线温湿度模组规格（炜盛Winsen V1.3）
## 一、产品概述
ZS03为数字单总线温湿度传感器，湿敏电阻+NTC测温，内置MCU，单线通信，低功耗小体积，适用于小家电、环境监测、仓储等场景。

## 二、关键参数
|项目|参数|
|----|----|
|型号|ZS03|
|供电电压|DC 2.7V~5.5V|
|湿度量程|20～90%RH|
|湿度精度|±5%RH（25℃、60%RH、标准电压）|
|温度量程|-20℃～60℃|
|测温精度|±1℃|
|工作湿度|20~90%RH（无凝露）|
|封装|SIP4 直插4引脚，引脚间距2.54mm|

## 三、引脚定义
```json
{
  "model": "ZS03",
  "pin_total": 4,
  "pin_info": [
    {"pin":1,"name":"VCC","func":"电源2.7~5.5V"},
    {"pin":2,"name":"DATA","func":"单总线双向数据脚"},
    {"pin":3,"name":"NC","func":"悬空空引脚，禁止接线"},
    {"pin":4,"name":"GND","func":"电源地"}
  ]
}
```
### 外围电路
DATA引脚外接**4.7K上拉电阻至VCC**；走线＞20m酌情减小上拉阻值；3.3V供电时导线建议≤1m。

## 四、单总线通信规范
### 1、一帧数据总长40bit（高位先发）
`湿度高8bit + 湿度低8bit + 温度高8bit + 温度低8bit + 校验8bit`
> 产品温、湿度小数固定=0；
> 校验规则：四个字节相加，取低8位=校验字节则数据有效。

### 2、温度正负判定
**温度低字节D7=1 → 负温度；D7=0 → 正温度**
> 备注：0℃以下湿度测量数据失效，仅温度可用。

### 3、整机通信时序
1. 空闲：总线默认高电平
2. 主机起始：拉低DATA **≥18ms**，发起读数
3. 释放总线，等待ZS03回复：传感器拉低80us作为应答；
   主机等待20~40us后读取应答电平，高电平代表无应答故障
4. 每1bit起始固定拉低50us：
   - 低电平保持26~28us → 数据0
   - 低电平50us、高电平70us → 数据1
5. 40bit接收完毕，传感器拉低50us，总线被上拉回到空闲。

## 五、读数使用规则
1. 单次读取返回**上次采样值**，需要实时数据连续读两次；
2. 两次读取间隔**≥5s**，频繁读取误差变大。

## 六、使用注意事项
1. 远离PCB发热元器件，保证通风，避免局部高温影响湿度精度；
2. 禁粉尘、腐蚀性气体环境长期使用，不可触碰内部湿敏芯片；
3. 仓储条件：10~40℃、＜60%RH，禁止凝露。

## 七、数据计算示例
### 示例1
数据：`0x37,0x00,0x12,0x00,0x49`
求和：0x37+0+0x12+0 = 0x49，校验正确
湿度=0x37=55%RH，温度=0x12=18℃（正温）

### 示例3
数据：`0x37,0x00,0x12,0x80,0xC9`
温度低字节最高位=1 → -18℃，校验匹配有效。
